何谓素材库？

当我们畅游网络世界的时候常常会发现很多好的、有趣的、有用的文章素材，这些素材包括文本，图片，视频等等。但是当自己写文章的时候，且想到使用这些素材的时候，我们往往会找不到这些素材或凭模糊的记忆满网络到处找，费时费力。为了避免这种情况的再一次发生而开发出的就是素材库。现在开始当你发现好的素材的时候，请不要犹豫！马上开动，把好的素材保存起来吧。一旦保存后当我们写相关主题文章的时候，就可以直接从素材库拖过来使用，省时省力，且写出好文章的概率也会大大提高。

安装小书签

要想搜集素材需得先安装小书签。在各个浏览器上的安装方法如下：

IE系列

  1. 首先把鼠标放到"素材搜集器"上，按下 "鼠标右键"。 键盘操作：先把鼠标焦点锁定到"素材搜集器"按钮上，再按下"Shift+F10"组合键即可。

  2. 在打开了的右键菜单中请选择"添加到收藏夹(F)"菜单。

  3. 当出现如下的安全警报对话框时请点击 "是(Y)"。

  4. 在"创建到(I)"项选择"收藏夹栏"(或"转到")文件夹后按下 "添加(A)"按钮。

  5. 如没有显示"收藏夹栏"(或 "转到")，请在标签栏点击"鼠标右键"后选择"收藏夹栏(A)"(或 "转到")。键盘操作：依次按下"Alt(菜单) - V(查看) - T(工具栏) - F(收藏夹栏)"键。

  6. 到此书签收集器已安装完毕。从此如发现好的素材，就可以点此小书签来收集素材了。键盘操作：先利用"Alt+C"组合键打开侧边收藏栏，之后选取收藏栏中的小书签来执行该操作。

  7. 查看页面时执行小书签就可以自动收集此页的素材。素材包括此页面地址，所有图片及视频代码。而且选取一段文本后再执行此小书签，可以直接将此段文本添加到引用及文本类型当中。注意：每次只能保存一种类型的素材。

Firefox

  1. 第一步，在素材搜集器按钮上按下"鼠标右键"。键盘操作： 先把鼠标焦点锁定到"素材搜集器"按钮上，再按下 "Shift+F10"组合键即可。

  2. 在弹出的右键菜单中选择"将此链接加为书签(L)"。

  3. 在文件夹项选择"书签工具栏"后，按下"保存"按钮。

  4. 如浏览器没有显示书签工具栏，请在标准工具栏按下鼠标右键后，选择"书签工具栏"。键盘操作：依次按下"Alt(菜单栏) - V(查看) - T(工具栏) - B(书签工具栏)"键。

  5. 到此书签收集器已安装完毕。从此如发现好的素材，就可以点此小书签来收集素材了。键盘操作：先利用"Ctrl(Cmd)+B"组合键打开侧边书签栏，之后选取书签工具栏中的小书签来执行该操作。

  6. 查看页面时执行小书签就可以自动收集此页的素材。素材包括此页面地址，所有图片及视频代码。而且选取一段文本后再执行此小书签，可以直接将此段文本添加到引用及文本类型当中。注意：每次只能保存一种类型的素材。

Safari

  1. 首先利用"Ctrl(Cmd)+Shift+B"组合键显示书签栏。

  2. 鼠标焦点准确放到素材搜集器"文本领域"后将此"拖放"到书签栏。

  3. 对弹出的书签名修改对话框，您可以修改书签名或直接点击"确定"来完成添加书签操作。

  4. 到此书签收集器已安装完毕。从此如发现好的素材，就可以点此小书签来收集素材了。键盘操作：先利用"Ctrl(Cmd)+Alt+B"组合键打开侧边书签栏，之后选取书签工具栏中的小书签来执行该操作。

  5. 查看页面时执行小书签就可以自动收集此页的素材。素材包括此页面地址，所有图片及视频代码。而且选取一段文本后再执行此小书签，可以直接将此段文本添加到引用及文本类型当中。注意：每次只能保存一种类型的素材。

  2. 鼠标焦点准确放到素材搜集器"文本领域"后将此"拖放"到书签栏。

  3. 到此书签收集器已安装完毕。从此如发现好的素材，就可以点此小书签来收集素材了。键盘操作：可利用"Ctrl(Cmd)+Shift+B"组合键打开书签管理器后，选取书签工具栏中的小书签来执行该操作。

  4. 查看页面时执行小书签就可以自动收集此页的素材。素材包括此页面地址，所有图片及视频代码。而且选取一段文本后再执行此小书签，可以直接将此段文本添加到引用及文本类型当中。注意：每次只能保存一种类型的素材。

Opera

  1. 首先，要显示书签侧栏。如没有显示请按下"Ctrl(Cmd)+B"组合键。

  2. 鼠标焦点锁定"素材搜集器"按钮，请按下"鼠标右键"。

  3. 鼠标右键菜单中选择"将链接加入书签(B)"。

  4. 到此书签收集器已安装完毕。从此如发现好的素材，就可以点此小书签来收集素材了。键盘操作：先利用"Ctrl(Cmd)+B"组合键打开侧边书签栏，之后选取书签工具栏中的小书签来执行该操作。

  5. 查看页面时执行小书签就可以自动收集此页的素材。素材包括此页面地址，所有图片及视频代码。而且选取一段文本后再执行此小书签，可以直接将此段文本添加到引用及文本类型当中。注意：每次只能保存一种类型的素材。

小书签使用方法

使用方法很简单，当发现好的素材的时候，点击小书签即可分门别类(文本，图片，链接，视频等)的收集各种素材。并且来源出处也随着各类素材自动保存。 提醒两点：1. 当您在有的页面根本无法收集素材的时候，您也不要埋怨，毕竟素材搜集器不是万能的噢。2. 收集素材的时候，一定要注意及尊重创作者的知识产权。
