글감 보관함이란?

우리 블로거들은 뉴스 사이트나 남의 블로그, 혹은 여타 웹 페이지에서 흥미로운 소재를 발견할 때가 많습니다. "이건 기억해뒀다가 나중에 관련 글을 쓸 때 써먹어야지" 하고 마음먹기도 하죠. 그러나 정작 글을 쓸 때면 그걸 어디서 봤는지 주소는 커녕 발견한 사실조차 기억나지 않을 때가 많습니다. 이제 좋은 글감을 발견하면 글감 보관함에 넣어 두세요. 수집해 놓은 내용(텍스트, 링크, 이미지, 동영상 등)은 글을 쓸 때 마우스로 끌어다가 손쉽게 첨부할 수 있습니다.

북마클릿 설치하기

글감을 수집하려면 사용하고 계신 브라우저에 글감 수집기 북마클릿을 설치해야 합니다. 북마클릿을 설치하는 방법은 브라우저 종류에 따라 조금씩 다릅니다. 아래 안내에 따라 사용중인 브라우저에 맞게 북마클릿을 설치하세요.

인터넷익스플로러

  1. 마우스 포인터를 글감 수집기 버튼 위치에 놓고 "마우스 오른쪽 버튼"을 누릅니다. 키보드 사용자는 글감 수집기 버튼에 포커스를 위치시킨 다음 "Shift+F10" 키를 누릅니다.

  2. 문맥 메뉴가 열리면 "즐겨찾기에 추가(F)"를 선택 합니다.

  3. 다음과 같은 보안 경고창이 뜨면 "예(Y)"를 선택합니다.

  4. 위치 지정 항목에서 "즐겨찾기 모음"(또는 "연결") 폴더를 선택한 다음 "추가(A)" 버튼을 누릅니다.

  5. "즐겨찾기 모음"(또는 "연결")이 화면에 표시되지 않았다면 탭 표시 영역에서 "마우스 오른쪽 버튼"을 누른 후 "즐겨찾기 모음(A)"(또는 "연결")을 클릭 합니다. 키보드 사용자는 "Alt(메뉴모음) - V(보기) - T(도구모음) - F(즐겨찾기 모음)" 키를 순차적으로 누릅니다.

  6. 글감 수집기 북마클릿을 사용할 준비가 됐습니다. 웹 서핑 도중 수집하고 싶은 글감이 있으면 언제든지 북마클릿을 클릭해 주세요. 키보드 사용자는 "Alt+C" 즐겨찾기메뉴를 이용해 북마클릿을 실행할 수 있습니다.

  7. 웹 서핑 도중 북마클릿을 실행하면 현재 페이지의 콘텐츠를 글감으로 저장할 수 있습니다. 인용하고 싶은 문장을 블럭으로 지정한 다음 북마클릿을 실행하면 텍스트 또는 인용구 형식으로 담을 수 있습니다. 현재 페이지의 이미지 목록이 자동으로 표시 됩니다. 현재 페이지의 동영상 삽입코드가 자동으로 포함 됩니다. 한 번에 한 가지 유형만 "글감 보관함에 보관하기" 할 수 있습니다.

파이어폭스

  1. 마우스 포인터를 글감 수집기 버튼 위치에 놓고 "마우스 오른쪽 버튼"을 누릅니다. 키보드 사용자는 글감 수집기 버튼에 포커스를 위치시킨 다음 "Shift+F10" 키를 누릅니다.

  2. 문맥 메뉴가 열리면 "링크를 북마크에 추가(L)"를 선택 합니다.

  3. 폴더 항목에서 "북마크 도구모음" 폴더를 선택한 다음 "완료" 버튼을 누릅니다.

  4. 북마크 도구모음이 화면에 표시되지 않았다면 브라우저의 표준 도구 모음 영역에서 마우스 오른쪽 버튼을 누른 후 "북마크 도구 모음"을 클릭 합니다. 키보드 사용자는 "Alt(표준 도구 모음) - V(보기) - T(도구모음) - B(북마크 도구모음)" 키를 순차적으로 누릅니다.

  5. 글감 수집기 북마클릿을 사용할 준비가 됐습니다. 웹 서핑 도중 수집하고 싶은 글감이 있으면 언제든지 북마클릿을 클릭해 주세요. 키보드 사용자는 "Ctrl(Cmd)+B"메뉴를 이용해 북마클릿을 실행할 수 있습니다.

  6. 웹 서핑 도중 북마클릿을 실행하면 현재 페이지의 콘텐츠를 글감으로 저장할 수 있습니다. 인용하고 싶은 문장을 블럭으로 지정한 다음 북마클릿을 실행하면 텍스트 또는 인용구 형식으로 담을 수 있습니다. 현재 페이지의 이미지 목록이 자동으로 표시 됩니다. 현재 페이지의 동영상 삽입코드가 자동으로 포함 됩니다. 한 번에 한 가지 유형만 "글감 보관함에 보관하기" 할 수 있습니다.

사파리

  1. 먼저 책갈피 막대를 화면에 표시해야 합니다. 책갈피 막대가 표시되지 않았다면 "Ctrl(Cmd)+Shift+B" 키를 누릅니다.

  2. 마우스 포인터를 글감 수집기 버튼의 '텍스트' 영역에 정확히 올려놓고 책갈피 막대로 "드래그 & 드롭" 합니다.

  3. 책갈피의 이름을 변경할 수 있는 대화 상자가 열렸습니다. 북마클릿의 이름을 변경하거나 또는 그냥 "승인" 버튼을 누릅니다.

  4. 글감 수집기 북마클릿을 사용할 준비가 됐습니다. 웹 서핑 도중 수집하고 싶은 글감이 있으면 언제든지 북마클릿을 클릭해 주세요. 키보드 사용자는 "Ctrl(Cmd)+Alt+B" 책갈피 막대 메뉴를 이용해 북마클릿을 실행할 수 있습니다.

  5. 웹 서핑 도중 북마클릿을 실행하면 현재 페이지의 콘텐츠를 글감으로 저장할 수 있습니다. 인용하고 싶은 문장을 블럭으로 지정한 다음 북마클릿을 실행하면 텍스트 또는 인용구 형식으로 담을 수 있습니다. 현재 페이지의 이미지 목록이 자동으로 표시 됩니다. 현재 페이지의 동영상 삽입코드가 자동으로 포함 됩니다. 한 번에 한 가지 유형만 "글감 보관함에 보관하기" 할 수 있습니다.

북마클릿 사용하기

이제 인터넷에서 좋은 글감을 발견하면 설치해둔 북마클릿을 클릭하세요. 텍스트, 이미지, 동영상 등 원하는 유형을 선택한 후 저장하시면 됩니다. 이렇게 수집한 글감들은 지금 보고 계신 글감보관함에 저장되며 글쓰기 화면에서 끌어다 쓸 수 있게 됩니다. 모든 글감은 자동으로 출처 정보와 함께 저장됩니다. 그러나 어떤 웹 사이트는 정보의 복제를 원천적으로 금하기도 합니다. 글감 수집기 북마클릿을 사용하실 때에는 타인의 지적재산권을 침해하는 일이 없도록 유의하시기 바랍니다.
